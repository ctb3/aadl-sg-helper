⚠️ Before you type your library password…

You should never enter your password for any website on a different website, but that's exactly what's happening here. Essentially, by doing so you're taking my word for it that I'm not stealing it. I promise I'm not! If you choose to continue, you're accepting that risk. But again, I don't know what I'd even do with your library password… rent a book? Check out the About section if you want to learn more about me.

If you'd rather not, that's completely fine too! Everything still works without an account. The app reads the code from your photo and hands you a prefilled aadl.org link so you can submit it yourself, signed in on your own browser. But in my opinion the fully automated mode is the most fun! ⚡

Your password passes through this app's server once to log in to aadl.org and is never stored or logged anywhere. Once logged in, aadl hands the session cookies back to my app, which stores them as normal in your device's localStorage. Every request the app makes to aadl from then on proxies those session cookies back to aadl. This lets the app stay signed in like the normal website, without this app having to sign in repeatedly or ever store passwords. You can see the code on my GitHub.

How it works

📷 Photograph a sign. Get as close as you can! Fill the frame with the sign, keep the code as straight as possible, and do your best to keep other signs or house numbers out of the shot.

🔍 The app reads the handwriting. Most codes are read quickly. Tricky ones get a closer look using Bedrock.

🏆 Connect your AADL account. If you log in to your AADL account, codes are submitted automatically for every player you check. This makes the majority of codes completely automatic once you snap the picture! Or if you don't want to log in, we'll still do our best to read it, and you'll get a prefilled aadl.org link to submit each code yourself.

✍️ If a read is wrong or AADL rejects it, you can always fix the code by hand.

📱 Add it to your home screen to open it full-screen like a real app — handy for snapping signs on the go.

In Safari, tap the Share button and then Add to Home Screen.

About

Hi! I'm Carl. 👋 I'm a fellow Ann Arborite and fan of the AADL Summer Game. I often pass signs while running or walking my dog and don't always want to stop and type in the code. Sometimes I'd just take a picture to enter later. My roots are in software development, so I always thought it'd be cool if there was an app that just read the sign and submitted the code automatically. Well, this is it! Having a computer read handwriting is a challenge, so it's not perfect. But it ended up working better than I hoped!

Note: this is a personal project, I'm not affiliated with the Ann Arbor District Library in any way. You should still hop over to aadl.org for all the other Summer Game content!
